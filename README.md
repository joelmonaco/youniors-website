# Youniors Website

Eine minimalistische Website für Youniors.

## Installation

```bash
npm install
```

## Entwicklung

```bash
npm run dev
```

Die Website läuft dann auf `http://localhost:8000`

## Struktur

- `index.html` - Hauptseite
- `server.js` - Einfacher HTTP-Server
- `fonts/` - Schriftarten (Vanguard CF und Suisse Intl)
